# Pharmacy Tools Hub v1.0

A polished, responsive, Vercel-ready landing page that brings five pharmacy tools into one unified gateway.

## Included Tools

- RxCalc
- SRSVault
- IDSLog
- DoseVault
- ExpiryRx

## Project Leaders

- FAISAL ALANAZI
- YARA ALOMARI

## Deployment

1. Extract the ZIP file.
2. Upload the folder to GitHub.
3. Import the repository into Vercel.
4. Deploy with the default static-site settings.

No build command or framework is required.
